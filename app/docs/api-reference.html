Sketch Tools

Fundamental tools for creating 2D sketches that form the basis for 3D models.

Line Tool

Draw straight lines in the sketch plane. Click two points to define start and end positions.

start
Point {x, y}
Starting point coordinates in sketch units (mm)
end
Point {x, y}
Ending point coordinates in sketch units (mm)
color
string
Line color (default: #58a6ff)
weight
number
Line thickness in pixels (default: 2)
Example: Create a horizontal line
draw.line({x: 0, y: 0}, {x: 100, y: 0})

Rectangle Tool

Create rectangular shapes by specifying width, height, and position.

width
number
Width in millimeters
height
number
Height in millimeters
origin
Point {x, y}
Bottom-left corner position (default: {0, 0})
fillColor
string
Fill color (default: none/transparent)
Example: 50×100mm rectangle
draw.rect({width: 50, height: 100})

Circle Tool

Draw circles with specified radius and center point.

radius
number
Circle radius in millimeters
center
Point {x, y}
Circle center coordinates (default: {0, 0})
fillColor
string
Fill color (default: none)
Example: 25mm radius circle
draw.circle({radius: 25, center: {x: 50, y: 50}})

Arc Tool

Create circular arc segments defined by start angle, end angle, and radius.

radius
number
Arc radius in millimeters
startAngle
number
Starting angle in degrees (0-360)
endAngle
number
Ending angle in degrees (0-360)
center
Point {x, y}
Arc center coordinates (default: {0, 0})
Example: 90° arc
draw.arc({radius: 30, startAngle: 0, endAngle: 90})

Polyline Tool

Draw connected line segments by specifying multiple points in sequence.

points
Point[]
Array of points [{x, y}, {x, y}, ...]
closed
boolean
Close the path (connect last to first point). Default: false
Example: Triangle
draw.polyline({ points: [{x: 0, y: 0}, {x: 50, y: 0}, {x: 25, y: 43}], closed: true })

3D Operations

Transform 2D sketches into 3D geometry with powerful modeling operations.

Extrude

Extend a 2D sketch into 3D space along the Z-axis by a specified distance.

sketch
Sketch
Sketch object to extrude
distance
number
Extrusion distance in millimeters
direction
string
Direction: 'up' | 'down' | 'both' (default: 'up')
taper
number
Taper angle in degrees (0-90, default: 0)
Example: Extrude rectangle to height
let sketch = draw.rect({width: 50, height: 50}); solid.extrude(sketch, {distance: 100})

Revolve

Rotate a 2D profile around an axis to create rotational geometry (like a lathe operation).

sketch
Sketch
Sketch profile to revolve
angle
number
Rotation angle in degrees (1-360, default: 360)
axis
string
Rotation axis: 'x' | 'y' | 'z' (default: 'y')
Example: Create a cone via revolve
let profile = draw.line({x: 0, y: 0}, {x: 20, y: 50}); solid.revolve(profile, {angle: 360, axis: 'y'})

Fillet

Round sharp edges with a smooth radius curve.

edges
Edge[]
Array of edges to fillet
radius
number
Fillet radius in millimeters
type
string
Fillet type: 'constant' | 'variable' (default: 'constant')
Example: Fillet all top edges
solid.fillet({edges: model.topEdges, radius: 2})

Chamfer

Create beveled edges with linear transition.

edges
Edge[]
Edges to chamfer
distance
number
Chamfer distance in millimeters
angle
number
Chamfer angle in degrees (15-60, default: 45)

Boolean Operations

Combine solids using set operations: Union, Subtract, Intersect.

Union

solid1
Solid
First solid to combine
solid2
Solid
Second solid to combine
Example: Combine two parts
let result = solid.union(part1, part2)

Subtract (Cut)

Remove the volume of one solid from another.

Example: Create a hole
let baseBlock = box({w: 100, h: 100, d: 50}); let hole = cylinder({r: 10, h: 50}); let result = solid.subtract(baseBlock, hole)

Intersect

Keep only the overlapping volume of two solids.

Example: Find intersection
let intersection = solid.intersect(solid1, solid2)

AI Commands

Natural language commands processed by cycleCAD's AI engine. Describe what you want to create in plain English.

Note: AI commands are converted to parametric models. Use specific dimensions and features for best results.

Supported Command Patterns

Basic Shapes

  • "50mm cube" → Creates a cubic box
  • "cylinder 30mm radius 60mm tall" → Rotational primitive
  • "sphere 25mm radius" → Spherical shape
  • "cone 40mm base 100mm tall" → Conical shape

Plates & Sheets

  • "plate 100 by 60 by 10" → Rectangular plate (W×H×D)
  • "plate 100x60x10 with 4 corner holes 8mm" → Plate with corner mounting
  • "sheet 500x300x2 with centered hole 25mm" → Perforated sheet

Brackets & Structural

  • "L bracket 80x40x5mm" → Right-angle bracket
  • "triangular gusset 60x60x5" → Reinforcement gusset
  • "mounting bracket 80mm base 60mm tall with 4 bolt holes" → Assembly mount

Mechanical Components

  • "spacer 20mm OD 10mm ID 5mm thick" → Ring spacer
  • "bushing 25mm outer 15mm inner 30mm long" → Cylindrical bushing
  • "bearing housing 80mm OD 50mm bore" → Flanged housing
  • "shaft collar 20mm bore 35mm OD 12mm wide" → Locking collar

Fasteners & Hardware

  • "M8 washer" → Standard flat washer
  • "hex standoff M4 15mm tall" → Threaded spacer
  • "bolt head M10 20mm long" → Hex bolt

Custom Features

  • "phone stand 80mm wide 60mm tall 5mm thick with slot" → Device holder
  • "cable clip for 10mm cable" → Cable organizer
  • "heat sink 80mm wide 40mm tall with 4 fins 3mm thick" → Cooling element
  • "filter housing 60mm diameter 80mm tall inlet outlet ports" → Canister body

AI Command Tips

  • Use specific dimensions (mm, mm, mm format)
  • Include material hints for realistic rendering
  • Mention mounting points (holes, pockets, slots)
  • Describe symmetry and repetition
  • AI fills in standard details (thread forms, standard sizes)

Export Formats

Save your designs in multiple industry-standard formats.

STL (Stereolithography)

Best for: 3D printing, CNC machining

format
string
'ascii' | 'binary' (binary is smaller)
scale
number
Scale factor (1.0 = original size)
Example: Export as binary STL
model.exportSTL({format: 'binary', scale: 1.0})

OBJ (Wavefront)

Best for: 3D visualization, rendering, animation

includeNormals
boolean
Include vertex normals (default: true)
includeMaterials
boolean
Export .mtl material file (default: true)
Example: Export as OBJ
model.exportOBJ({includeNormals: true, includeMaterials: true})

STEP/IGES (CAD Interchange)

Best for: Professional CAD exchange, parametric history

preserveHistory
boolean
Keep parametric feature tree (default: true)
Example: Export as STEP
model.exportSTEP({preserveHistory: true})

glTF 2.0

Best for: Web display, AR/VR, game engines

embedded
boolean
Embed textures in .glb file (default: true)
compressed
boolean
Use Draco compression (default: true)

PLY (Polygon File Format)

Best for: Point cloud data, mesh analysis

format
string
'ascii' | 'binary' (binary recommended)
includeColors
boolean
Include vertex colors (default: true)

JSON (Native Format)

Best for: Saving project state, parametric recovery

compressed
boolean
GZIP compression (default: true)
Example: Save project
model.exportJSON({compressed: true})
Format File Size Editability Best Use
STL Medium No 3D Printing

STEP Large Yes CAD Exchange
glTF Small No Web/AR
PLY Medium No Analysis

URL Parameters

Deep-link to cycleCAD with pre-configured models and parameters.

Basic URL Structure

https://explodeview.com/cyclecad?create=box&w=50&h=30&d=20

Global Parameters

create
string
Primitive type: box, cylinder, sphere, cone, torus
prompt
string
AI natural language prompt (URL encoded)
material
string
Material preset: steel, aluminum, plastic, etc.
color
string
Hex color code (e.g., #ff0000 for red)

Box Parameters

?create=box&w=100&h=60&d=20&material=aluminum&color=#cccccc
w
number
Width in mm
h
number
Height in mm
d
number
Depth in mm

Cylinder Parameters

?create=cylinder&r=30&h=60
r
number
Radius in mm
h
number
Height in mm

Sphere Parameters

?create=sphere&r=25
r
number
Radius in mm

AI Prompt URL

?prompt=plate%20100x60x10%20with%204%20corner%20holes%208mm
Important: Spaces and special characters must be URL-encoded. Use %20 for space.
