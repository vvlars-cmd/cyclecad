Getting Started with cycleCAD

Learn browser-based parametric 3D modeling in minutes. No installation required.

1 Welcome to cycleCAD

cycleCAD is a browser-based parametric 3D modeler powered by OpenCascade. Create professional-grade CAD models directly in your browser without installing any software.

What You Can Do

  • Sketch & Extrude: Draw 2D sketches and extrude them into 3D parts
  • AI Text-to-3D: Describe a part in natural language and watch it generate instantly
  • Import STEP Files: Load existing CAD designs and modify them parametrically
  • Export Multiple Formats: STL (3D printing), OBJ, STEP, and cycleCAD JSON
  • Parametric Design: Change dimensions anytime and the model updates automatically
💡 Zero Installation

cycleCAD runs entirely in your browser. Just open it, start creating, and your models sync to the cloud. Work from any device, any browser.

Three Ways to Create

Method Best For Skill Level
Sketch → Extrude Precise parts with exact dimensions Beginner to Advanced
AI Text-to-3D Quick prototyping, exploration, ideas Beginner
Import & Modify Iterating on existing designs Intermediate

2 Quick Start (60 Seconds)

Follow these steps to create your first 3D part. Estimated time: 60 seconds.

1 Open cycleCAD in your browser
Navigate to https://cyclecad.com or open it locally. You'll see the welcome screen.
2 Click "New Sketch" on the welcome screen
A blank canvas appears. You're now in sketch mode, ready to draw 2D shapes.
3 Select Rectangle tool (press R)
The rectangle tool is active. The cursor changes to a crosshair. You can also click the Rectangle icon in the toolbar.
4 Click two points to draw a 50×30mm rectangle
Click once for the first corner. A dimensions panel appears. Type 50 for width and 30 for height. Click the second corner to complete the rectangle.
5 Press Enter to finish the sketch
The sketch is now complete. The 2D shape is ready to be extruded into 3D.
6 Click Extrude (or press E)
The extrude tool is activated. You can now define the height of your 3D part.
7 Enter 20mm height → You have your first 3D part!
Type 20 in the height field. Press Enter. Congratulations! You've created a 50×30×20mm solid block. The 3D view shows your part with lighting and shadows.
✓ You Did It!

You just created your first parametric 3D model. In the Properties panel on the right, you can click any dimension to edit it and see your model update in real-time. Try changing the width to 60mm—the entire part updates instantly.

3 Tutorial: Creating a Mounting Bracket

A practical walkthrough creating a mounting bracket with bolt holes—a common part in mechanical design.

Step 1: Create the Base Sketch

  1. Click New Sketch (or press Ctrl+N)
  2. Select the Rectangle tool (R key)
  3. Draw a rectangle: 80mm wide × 40mm tall
  4. Press Enter to finish the sketch
┌──────────────────────────────┐ │ │ 40mm │ BRACKET │ │ BASE │ │ │ └──────────────────────────────┘ 80mm

Step 2: Add Bolt Hole Positions

  1. Press S to start a new sketch on the base
  2. Use the Circle tool (C key) to place 4 circles
  3. Position circles at corners: 8mm diameter each
    • Top-left: 10mm from edges
    • Top-right: 10mm from edges
    • Bottom-left: 10mm from edges
    • Bottom-right: 10mm from edges
  4. Press Enter when done
•═══════════════════════════════• ║ ⊙ ⊙ ║ ║ ║ 40mm ║ ║ ║ ⊙ ⊙ ║ •═══════════════════════════════• 80mm ⊙ = 8mm bolt holes (4 total) • = Corner reference

Step 3: Extrude to 3D

  1. Click the base rectangle (now your 2D sketch)
  2. Press E to extrude
  3. Enter height: 5mm
  4. The bracket is now 3D with bolt holes cut through

Step 4: Finishing Touches (Optional)

  1. Select all edges (Ctrl+A)
  2. Add a Chamfer (0.5mm on edges) for a polished look
  3. Press M to switch to move/rotate mode
  4. Rotate the part to inspect it from all angles
🔧 Parametric Magic

Click any dimension in the Properties panel on the right (width, height, hole diameter, etc.) to edit it. Change the hole diameter to 6mm? The entire part updates instantly. This is parametric design—perfect for iterating designs quickly.

4 Tutorial: AI-Generated Part

Let AI do the heavy lifting. Describe a part in plain English, and cycleCAD generates it in seconds.

Step 1: Open AI Chat

  1. Click the AI Chat button in the right panel (or press ?)
  2. A chat window opens on the right side of the screen

Step 2: Describe Your Part

Type a natural language description in the chat box:

Create a flanged bearing housing: - 80mm outer diameter - 50mm bore (center hole) - 4 bolt holes at 60mm bolt circle diameter - Material: Aluminum - Flange thickness: 8mm

Step 3: AI Generates Instantly

  1. Press Enter or click Send
  2. The AI analyzes your description in real-time
  3. Within 2-3 seconds, a 3D model appears in the viewport
  4. The part is fully parametric—dimensions appear in the Properties panel

Step 4: Customize the Generated Part

  1. Look at the Properties panel on the right
  2. Click any parameter (outer diameter, bore, bolt hole count, etc.)
  3. Edit the value and press Enter—the 3D model updates instantly
  4. Example: Change bore from 50mm to 55mm. Watch the model update in real-time.
🤖 Be Specific, Be Clear

The more details you provide, the better the AI understands your intent. Include dimensions, materials, hole patterns, and any special features. Vague descriptions like "make a cool housing" may not generate what you expect. Specific descriptions like "cylindrical housing, 50mm OD, 25mm ID, 30mm height, 4 evenly-spaced M3 bolt holes" work best.

Example Prompts That Work Well

  • "50mm cube made of steel"
  • "Cylinder: 30mm radius, 60mm height, hole 8mm down the center"
  • "Plate 100×60×10mm with 4 corner holes 8mm diameter"
  • "M8 washer (standard metric)"
  • "Gear: 40mm diameter, 20 teeth, 5mm bore"
  • "L-bracket: 50mm legs, 10mm thick, 4 corner holes"

5 Tutorial: Complex Part (Spacer with Features)

Create a more advanced part: a cylindrical spacer with a center bore and chamfered edges.

Step 1: Create the Cylinder Base

  1. Press Ctrl+N for a new sketch
  2. Select Circle tool (C key)
  3. Draw a circle: 30mm diameter (15mm radius from center)
  4. Press Enter to finish the sketch

Step 2: Extrude to Create Cylinder

  1. Press E to extrude
  2. Enter height: 15mm
  3. You now have a solid cylinder: 30mm diameter × 15mm tall

Step 3: Add Center Bore

  1. Click the top face of the cylinder (or press F for face select)
  2. Press S to create a new sketch on the top face
  3. Draw a circle: 10mm diameter
  4. Press Enter to finish
  5. Press P for Pocket (subtractive extrude)
  6. Enter depth: 15mm (through the entire part)
Side View: Top View (after bore): ┌──────────────┐ │ │ ┌────────────────┐ │ SPACER │ │ ○ │ │ │ │ 30mm OD │ │ │ │ 10mm ID │ └──────────────┘ └────────────────┘ 15mm tall

Step 4: Add Chamfer to Top Edge

  1. Select the top circular edge (top rim of the bore)
  2. Press Shift+C for Chamfer
  3. Enter chamfer distance: 1mm
  4. The sharp edge is now beveled

Step 5: Export as STEP

  1. Click the Export button in the top toolbar
  2. Select STEP (.step) format
  3. Name your part: spacer_v1.step
  4. Click Download
  5. The file is saved to your computer and can be opened in any CAD software
🎯 Features vs. Sketches

In cycleCAD, Features are the operations you apply (Extrude, Pocket, Chamfer, Fillet, etc.). Each feature modifies the geometry step-by-step. The Feature Tree on the left shows every operation—click any feature to edit it or use it as a reference.

6 Interface Guide

Understand the cycleCAD workspace. Every panel has a purpose.

Left Panel: Feature Tree

Shows the history of all operations you've performed on your model (also called the "Feature Tree" or "Model Tree").

  • Sketch1, Sketch2, …: 2D sketches you've created
  • Extrude1, Pocket1, …: 3D features applied to sketches
  • Chamfer1, Fillet1, …: Finishing operations (edge treatments)
  • Expand/collapse: Click arrows (▶ ▼) to show/hide sketch details
  • Edit: Double-click any feature to edit it
  • Delete: Right-click a feature and select Delete to remove it and all dependent features
  • Visibility: Click the eye icon next to a feature to hide/show it

Center: 3D Viewport

Your main canvas. This is where you see and manipulate your 3D model in real-time.

  • Left-click + drag: Rotate the model (arcball rotation)
  • Right-click + drag: Pan (move the model side-to-side)
  • Scroll wheel: Zoom in and out
  • Middle-click: Reset the view to default (fit all)
  • Press . (period): Fit selected part in view
  • Click a face/edge/vertex: Select it (highlighted in blue/orange)

Right Panel: Properties & AI Chat

Two tabs: Properties and AI Chat.

Properties Tab:

  • Shows all editable dimensions of the selected feature
  • Examples: width, height, depth, radius, hole diameter, chamfer distance
  • Click any value to edit it. Press Enter to apply changes.
  • Material selector: Choose material (Steel, Aluminum, Plastic, etc.) at the bottom
  • Mass estimation: Calculated based on material density and bounding volume

AI Chat Tab:

  • Describe parts in plain English
  • Ask questions about your model ("How much does this weigh?", "Can I 3D print this?")
  • Chat history is saved for each project

Top Toolbar: Tools Organized by Category

Bottom Status Bar

  • Current mode: "Sketch Mode", "Feature Mode", "View Mode", etc.
  • Units: Millimeters (mm) by default. Change in Preferences (Ctrl+,)
  • Grid: Grid spacing and snap settings
  • Selected element count: "1 face selected", "3 edges selected", etc.
  • Coordinate display: Mouse pointer position in model space
📐 Coordinate System

cycleCAD uses a standard right-handed 3D coordinate system. The red axis is X (left-right), green is Y (forward-back), and blue is Z (up-down). You can toggle the axes display with Ctrl+Shift+A.

⚠️ AI Limitations

The AI is good at standard, geometric parts but may struggle with complex organic shapes, highly intricate details, or custom non-standard geometries. For those cases, use the sketch → extrude workflow manually. Also, the AI generates "good enough" geometry—tolerances and thread details are simplified for speed.

Pro Tips for AI Prompts

  • Be specific with dimensions: "50mm cube" works better than "a small cube"
  • Specify material if it matters: "Aluminum flange" vs. "Steel flange" (affects density/weight)
  • Use metric units: Millimeters (mm), Meters (m). Avoid inches.
  • Describe standard parts by their name: "M8 bolt", "DIN 125 washer", "ANSI flange" → AI recognizes these
  • Avoid ambiguous descriptions: "Make a cool housing" might not give you what you expect. Say "Cylindrical housing, 60mm OD, 40mm ID, 80mm tall, 4 mounting holes"
  • Iterate if needed: If the first result isn't perfect, ask the AI to modify: "Add a chamfer to all edges", "Make the bore 2mm larger", "Change material to aluminum"

9 Export Options

cycleCAD supports multiple export formats for different workflows. Export your design for 3D printing, CAD software, manufacturing, or sharing.

STL (Stereolithography)

Format Best For Details
.stl (Binary or ASCII) 3D printing, slicing software Triangulated mesh. 0.1mm resolution. Supported by Cura, PrusaSlicer, Ultimaker, Formlabs, etc. Binary is smaller; ASCII is human-readable.

How to export STL:

  1. Click Export in the top toolbar
  2. Select STL (.stl)
  3. Choose Binary or ASCII (Binary is recommended—smaller file size)
  4. Enter filename (e.g., bracket_v1.stl)
  5. Click Download
  6. Import into your slicer software for 3D printing

OBJ (Wavefront)

Format Best For Details
.obj 3D software (Blender, Maya, Cinema 4D), game engines Polygon mesh format. Includes vertex normals for smooth shading. Supported by most 3D software. Files are larger than STL.

How to export OBJ:

  1. Click Export in the top toolbar
  2. Select OBJ (.obj)
  3. Enter filename
  4. Click Download
  5. Imports into Blender, Maya, etc. for texturing, rendering, or further editing

STEP (ISO 10303-21)

Format Best For Details
.step (also .stp) CAD software (Fusion 360, SolidWorks, FreeCAD), manufacturing Parametric geometry interchange. Preserves features and constraints. Best for re-opening and editing in other CAD programs. Requires OpenCascade kernel.

How to export STEP:

  1. Click Export in the top toolbar
  2. Select STEP (.step)
  3. Enter filename (e.g., bracket_v1.step)
  4. Click Download
  5. Open in any STEP-compatible CAD software for further modification

cycleCAD JSON (Project Archive)

Format Best For Details
.cyclecad.json Saving and reopening projects in cycleCAD Stores complete project data: sketches, features, parameters, materials, history. Re-open and edit parametrically. Only openable in cycleCAD.

How to export cycleCAD JSON:

  1. Click Export in the top toolbar
  2. Select cycleCAD Project (.cyclecad.json)
  3. Click Download
  4. To reopen: Click Open Project and select the .cyclecad.json file
  5. All sketches, features, and parameters are restored—full editability

PDF (Technical Drawing)

Format Best For Details
.pdf Documentation, sharing blueprints, manufacturing specs 2D orthographic projections (front, top, side views) with dimensions. Includes BOM (Bill of Materials).

How to export PDF:

  1. Click Export in the top toolbar
  2. Select Drawing (PDF)
  3. Choose views to include: Front, Top, Side, Isometric (check as needed)
  4. Include BOM? (optional)
  5. Click Generate PDF
  6. PDF opens in a new tab—save or print as needed
📤 Export Best Practices
  • 3D Printing? Use STL. Binary STL is preferred (smaller files).
  • Sharing with collaborators in other CAD tools? Use STEP for full editability.
  • Sending to Blender/Maya for rendering? Use OBJ.
  • Want to keep working on this project later? Export as cycleCAD JSON (preserves parametric features).
  • Manufacturing or blueprint sharing? Export as PDF for 2D drawings + BOM.

Batch Export (Export Multiple Parts)

If your project has multiple bodies or assemblies:

  1. Click Export in the top toolbar
  2. Select Batch Export All Parts
  3. Choose format(s): STL, OBJ, STEP, or a combination
  4. Click Download All
  5. A zip file is created with one file per part, properly named

Comparison: Which Format Should I Use?

Use Case Format Reason
3D Printing (Cura, PrusaSlicer) STL Industry standard, smallest files, all slicers support it
Render in Blender/Maya OBJ or STEP OBJ for mesh editing, STEP for parametric re-import
Edit in SolidWorks/Fusion 360 STEP Preserves history and parameters
Send to manufacturer (CNC/injection mold) STEP Full geometry, no approximations
Continue editing in cycleCAD later cycleCAD JSON Preserves all sketches, features, full editability
2D Blueprint + BOM for docs PDF Orthographic views, dimensions, parts list
💾 Pro Tip: Always Keep the cycleCAD JSON

When you export a project, always also save a copy as cycleCAD JSON. This preserves all your parametric features, sketches, and history. If you ever need to make changes, you can reopen the JSON file in cycleCAD and edit parametrically—much faster than redrawing from an STL or STEP file.
