Rebuild Guide: Leistenbürstenblech (Strip Brush Sheet Metal)

1
Create New Sheet Metal Design

✓ Fusion 360

  1. File → New Design
  2. In the Ribbon, click the SHEET METAL tab
  3. Click New Sheet Metal Design button
  4. Project is now in Sheet Metal workspace
  5. Units should be Millimeters (metric)

⚙ cycleCAD API

const design = new cycleCAD.Design({
  name: 'Leistenbürstenblech',
  type: 'SHEET_METAL',
  units: 'mm',
  inventor_source: {
    filename: 'Leistenbürstenblech.ipt',
    version: '11.0',
    template: 'Sheet Metal (mm)'
  }
});
💡 Tip: Sheet Metal workspace has different tools than Part Design. You'll see Base Flange, Flange, Bend, and Flat Pattern commands instead of Pad/Pocket.
2
Set Sheet Metal Rules (Thickness & Bend)

✓ Fusion 360

  1. Right-click on Sheet Metal folder in Browser
  2. Select Edit Sheet Metal Rules
  3. Set Thickness: 1.5 mm (or from parameter)
  4. Set Bend Radius: 2.0 mm (design-dependent)
  5. Set Bend Relief: check if design requires relief
  6. Click OK

⚙ cycleCAD API

design.setSheetMetalRules({
  thickness: 1.5, // mm
  bendRadius: 2.0,
  bendRelief: 'none', // 'none'|'rectangular'|'obround'
  gapSize: 0,
  cornerStyle: 'sharp' // or 'rounded'
});
💡 Tip: The Stärke parameter from the original Inventor file is the thickness value. Look for this in the Inventor properties or constraints.
3
Define User Parameters

✓ Fusion 360

  1. Click Modify → Change Parameters
  2. In User Parameters panel, click Create
  3. Add parameter Stärke (Thickness)
    • Type: Length
    • Default: 1.5 mm
    • Expression: =Sheet_Metal_Thickness
  4. Add other parameters as needed (width, length, etc.)
  5. Click OK

⚙ cycleCAD API

design.parameters = {
  Stärke: {
    type: 'length',
    value: 1.5,
    unit: 'mm',
    source: 'Skizze.ipt',
    parameterId: 11
  },
  // Add other parameters here
};
💡 Tip: The original Inventor file references Skizze.ipt (the master parameter sketch). Extract all parameters from there and recreate them in Fusion 360.
4
Create Base Sketch (Strip Brush Profile)

✓ Fusion 360

  1. Click Sheet Metal tab → New Sketch
  2. Select XY Plane (Origin)
  3. Sketch the strip brush profile:
    • Draw the flat rectangle profile (strip shape)
    • Constrain dimensions using Stärke parameter
    • Add width, length, and any notches/holes for mounting
  4. Apply all constraints (fully constrained = blue sketch)
  5. Click Finish Sketch

⚙ cycleCAD API

const sketch = design.createSketch('BaseProfile', 'XY');
sketch.addRectangle({
  width: 45, // mm
  height: 150, // mm
  constrainToParam: 'Stärke'
});
sketch.addHole({
  x: 22.5,
  y: 75,
  diameter: 6
});
sketch.finalize();
💡 Tip: The Leistenbürstenblech is a brush holder strip. Look at the original Inventor sketch to understand the exact profile shape, mounting holes, and bristle attachment points.
5
Create Base Flange from Sketch

✓ Fusion 360

  1. Select the BaseProfile sketch in the Browser
  2. Click Sheet Metal tab → Base Flange button
  3. In dialog:
    • Confirm the sketch is selected
    • Height: 0 mm (flat sheet)
    • Material Side: Outside or Inside (depends on design)
  4. Click OK
  5. The base flange is now your flat sheet

⚙ cycleCAD API

const baseFlange = design.createBaseFlange({
  sketch: 'BaseProfile',
  height: 0,
  materialSide: 'inside',
  thickness: design.parameters.Stärke.value
});
💡 Tip: If the original Inventor file has a FlatPattern feature, this design may already be flat. In that case, the Base Flange is the starting point for adding bends/flanges if needed.
6
Add Bends & Flanges (if design requires)

✓ Fusion 360

  1. If the design has bends/folds:
    • Click Sheet Metal → Flange
    • Select the edge where bend should occur
    • Set angle (typically 90°) and flange width
  2. Or use Bend for existing surfaces
  3. Repeat for each bend needed
  4. Verify the 3D shape matches original Inventor design

⚙ cycleCAD API

// Add flange (for bending edges)
baseFlange.addFlange({
  edge: 'edge_0',
  angle: 90,
  flangeWidth: 15,
  bendRadius: 2.0
});

// Or add bend on existing surface
baseFlange.addBend({
  face: 'face_0',
  bendLine: 'line_0',
  angle: 45,
  radius: 2.0
});
💡 Tip: For the Leistenbürstenblech, check if it has any bends. If it's a simple flat strip, skip this step. If it has folded edges or flanges, add them here. Review the original Inventor 3D view to confirm.
7
Create Flat Pattern

✓ Fusion 360

  1. Click Sheet Metal → Flat Pattern button
  2. Select the face to be primary (usually the base flange)
  3. Fusion 360 automatically unfolds all bends
  4. Verify all dimensions unfold correctly:
    • Check bend allowances are calculated
    • Measure overall width/length of unfolded pattern
  5. The FlatPattern feature now appears in Browser
  6. You can toggle between folded and flat view

⚙ cycleCAD API

const flatPattern = design.createFlatPattern({
  primaryFace: 'baseFlange',
  bendAllowanceMethod: 'auto',
  bendDeduction: 0, // mm (auto-calculated)
  name: 'FlatPattern'
});

// Get flat pattern dimensions for laser cutting
const flatDims = flatPattern.getDimensions();
console.log('Width:', flatDims.width, 'mm');
console.log('Height:', flatDims.height, 'mm');
💡 Tip: This is the most important step for sheet metal. The flat pattern shows exactly how the part looks when cut flat before folding. Save this view — you'll need it for laser cutting (DXF export).
8
Export DXF for Laser Cutting

✓ Fusion 360

  1. Make sure the FlatPattern feature is active (highlighted in Browser)
  2. Right-click on FlatPattern → Save as DXF
  3. Choose export options:
    • Format: DXF R14 (2D)
    • Units: Millimeters
    • Scale: 1:1
  4. Save as: Leistenbürstenblech_FlatPattern.dxf
  5. This DXF is ready for laser cutting software (LightBurn, CorelDraw, etc.)

⚙ cycleCAD API

flatPattern.exportDXF({
  filename: 'Leistenbürstenblech_FlatPattern.dxf',
  format: 'DXF_R14',
  units: 'mm',
  scale: 1.0,
  includeHoles: true,
  includeBendLines: true,
  bendLineStyle: 'dashed'
});

console.log('✓ DXF exported successfully');
💡 Tip: The DXF file can be imported into laser cutter software. Make sure the scale is 1:1 (100%) and units are in millimeters.
9
Verify Dimensions Against Original

✓ Fusion 360

  1. Open both the Fusion 360 model and original Inventor file side-by-side
  2. Check critical dimensions:
    • Thickness (Stärke): 1.5 mm
    • Width: verify against original
    • Length: verify against original
    • Hole positions: check X/Y coordinates
    • Bend radius: match original bend settings
  3. Use Inspect → Measure tool for precision checks
  4. Compare 3D folded model to original Inventor 3D view
  5. Verify flat pattern dimensions for laser cutting accuracy

⚙ cycleCAD API

// Verification checklist
const verification = {
  thickness: 1.5,
  originalFileName: 'Leistenbürstenblech.ipt',
  parameterId: 11,
  dimensions: {
    width: { measured: 45, tolerance: 0.1 },
    length: { measured: 150, tolerance: 0.1 },
    bendRadius: { measured: 2.0, tolerance: 0.2 }
  },
  features: {
    hasHoles: true,
    hasBends: false,
    flatPatternReady: true
  }
};

console.log('✓ All dimensions verified');
💡 Tip: Keep a verification checklist. Compare Fusion 360 measurements with the original Inventor part properties. Small discrepancies (±0.1 mm) are acceptable due to rounding, but larger gaps indicate a rebuild error.

Save & Export Final Design

✓ Fusion 360

  1. Click File → Save (or Ctrl+S)
  2. Export formats:
    • STEP (.step): 3D model for CAD exchange
    • IGES (.iges): Legacy CAD format
    • STL (.stl): 3D printing (if needed)
    • DXF (.dxf): Already done (flat pattern)
  3. Right-click design → Save as type...
  4. Save as: Leistenbürstenblech.step

⚙ cycleCAD API

// Save design in all formats
design.save('Leistenbürstenblech', {
  formats: ['step', 'dxf', 'iges', 'stl'],
  metadata: {
    originalFile: 'Leistenbürstenblech.ipt',
    originalVersion: '11.0',
    rebuildDate: '2026-03-24',
    software: 'Fusion 360',
    status: 'verified'
  }
});

console.log('✓ Design saved in all formats');
console.log('Ready for manufacturing!');
💡 Tip: Keep both the parametric Fusion 360 design (.f3d) and exported formats (STEP, DXF) for future reference and manufacturing.
