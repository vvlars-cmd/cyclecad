Quick Start Guide

1. Create shapes
Use Chat or toolbar: "box 50mm", "cylinder r20 h40", "gear 60mm 24 teeth"
2. Modify parts
"move it up 20", "rotate 45", "scale 2x", "fillet 5mm", "reduce height to 30"
3. Boolean operations
"subtract box from cylinder", "intersect", "union"
4. Scene management
"delete it", "undo", "hide it", "show all", "clear scene"
5. Sketch mode
Click Sketch in toolbar, draw with line/rect/circle/arc, then extrude
6. Export
"export stl", "export obj", "export dxf", "export gltf"
7. Keyboard shortcuts
S=Sketch, E=Extrude, F=Fillet, C=Chamfer, G=Grid, W=Wireframe, Del=Delete, ?=Help
AI Chat
Type natural language in the Chat tab. Understands typos: "cylindr", "interset", "subtrat"
Import
Supports Inventor .ipt/.iam, STL, OBJ, and STEP (via server)
Agent API
window.cycleCAD.execute({ method: "shape.cylinder", params: { radius: 25, height: 60 } })

ExplodeView Integration

Viewer Mode
Press V or click Viewer Mode to switch to ExplodeView. All 57 viewer features are available inside cycleCAD — shared 3D scene, no duplication.
Available in Viewer Mode

Viewer API
