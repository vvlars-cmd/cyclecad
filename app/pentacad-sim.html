Pentacad Simulator — quick help

    
Browser-native 5-axis G-code simulator for Pentamachine V1 / V2 / Solo. Everything runs locally; no file is uploaded anywhere.


    
Keyboard shortcuts

    

      
Space — play / pause

      
→ / ← — step one move forward / back

      
Home — go to start

      
End — jump to last move

      
F — fit view

      
R — reset camera (isometric)

      
1 / 3 / 7 — front / right / top view

      
Cmd/Ctrl + O — open file

      
Cmd/Ctrl + E — examples

    


    
Supported G/M codes

    

      Motion: G0 G1 G2 G3 G28 G38.2

      Modal: G17/18/19 G20/21 G40 G54-59 G90/91 G93/94/95

      Canned: G81 G82 G83 G73

      M: M0 M1 M2 M3 M4 M5 M6 M7 M8 M9 M30
