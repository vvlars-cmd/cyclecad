Pentacad Simulator — Help

    
Pentacad Simulator plays back 5-axis G-code on a virtual Pentamachine. Drop a .ngc / .nc / .gcode file anywhere on the page, or use the toolbar to load one.


    
Workspaces

    
    
      
  • GCODE — syntax-highlighted G-code with line numbers, warning markers (⚠) for unimplemented codes, info markers (ℹ) for parser observations.
    • 
      
  • SUMMARY — diagnostics, runtime estimate, active tool, per-tool fields (Z offset, diameter, holder).
    • 
    


    
Viewport widgets

    
    
      
  • ViewCube — drag to orbit, click a face to snap.
    • 
      
  • Locate — re-centers the camera on the machine.
    • 
      
  • Screenshot — saves the current viewport as PNG.
    • 
      
  • DRO — live X/Y/Z/A/B/T readout during playback.
    • 
      
  • Show Options — opens the Back Plot panel for tool-path range, automatic update, time-window playback.
    • 
      
  • Change Machine — switch between Solo, V1, V1 Kickstarter, V2-10, V2-50.
    • 
    


    
Keyboard shortcuts

← →

Home End
jump to start / end

open G-code file

screenshot
