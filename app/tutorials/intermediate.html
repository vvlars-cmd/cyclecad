Advanced Sketch Techniques
Constraints, geometry tools, and parametric sketching

Fully constrained sketches are the foundation of parametric modeling. cycleCAD's constraint solver supports 12 constraint types that define relationships between geometry.

🔗 Constraint Types
Constraint Description Use Case
Coincident Two points or endpoints overlap Align circles at origin
Horizontal Line parallel to X-axis Level edges
Vertical Line parallel to Y-axis Plumb edges
Parallel Two lines stay parallel Slot geometry
Perpendicular Two lines are 90° Right angles
Tangent Line touches curve smoothly Rounded corners
Equal Lengths or radii match Symmetric parts
Concentric Circles share same center Hole placement
Symmetric Mirrored across centerline Bilateral symmetry
Distance Fixed distance between points Dimension spacing
Angle Fixed angle between lines Angled edges
Fixed Lock point or geometry in place Reference geometry
Pro Tip: Constraint Strategy
Start with geometric constraints (coincident, parallel, perpendicular) before adding dimensional constraints (distance, angle). This reduces solver iterations and creates more robust designs.
📐 Advanced Geometry Tools

Spline B — Draw smooth curves through multiple points. Adjust control points post-draw.

Polygon Shift+P — Regular polygons (3-sided to 12-sided). Set center and radius.

Text T — Add sketch text (NOT extruded text). Useful for labeling drawings.

Ellipse E — Elliptical geometry. Click center, then drag major/minor axes.

Slot Shift+S — Rounded rectangle. Specify width, length, and corner radius.

✂️ Sketch Modifiers
Modifier Shortcut Description
Trim Shift+T Remove line segments between intersections
Extend Shift+E Stretch line to nearest edge
Split Shift+L Break curve at intersection point
Offset O Parallel copy at fixed distance
Mirror M Flip geometry across centerline
Move/Copy Shift+M Duplicate while translating
Scale Shift+K Enlarge/shrink from center point
Watch Out
Over-constraining a sketch (redundant constraints) will block solver. The status bar shows "3 DOF" (degrees of freedom). Fully constrained = 0 DOF. If stuck, delete the last constraint and re-apply.

Parametric Operations
Fillet, chamfer, boolean, shell, and patterns
🔲 Fillet

Round edges with a smooth curve. Select one or more edges, then specify radius. Fillets adapt when you change upstream sketches or extrusion heights.

  • Single-radius fillet: Constant radius all edges
  • Variable fillet: Click edge, set custom radius in properties
  • Multi-edge: Shift-click multiple edges, apply one operation
🔪 Chamfer

Cut a 45° or angled bevel on edges. Useful for deburring and assembly fit.

  • Distance chamfer: Equal cut length on both faces (1mm)
  • Angled chamfer: Specify angle and one distance (2mm @ 30°)
🔄 Boolean Operations

Combine multiple solid bodies (union, cut, intersect).

Operation Shortcut Result
Union Ctrl+U Merge two bodies into one
Cut Ctrl+X Remove tool body from target
Intersect Ctrl+I Keep only overlapping volume
🏠 Shell

Create hollow geometry by removing one or more faces and setting wall thickness (e.g., 2mm).

  • Select faces to remove
  • Set uniform wall thickness
  • Ideal for plastic injection molding
📦 Rectangular Pattern

Array features in X and Y directions. Select a feature (fillet, hole, boss), then set count and spacing.

// Example: 3×4 array of holes, 10mm spacing Pattern: Type=Rectangular, Count-X=3, Spacing-X=10mm, Count-Y=4, Spacing-Y=10mm
⭕ Circular Pattern

Array features in a circle around a center point. Useful for bolt holes, turbine blades.

  • Set count (8 holes)
  • Set radius (50mm from center)
  • Optional: rotation offset
Pro Tip: Pattern Editing
Once created, patterns are "smart." Change the original feature, and all array instances update automatically.

Advanced Operations
Sweep, loft, sheet metal, springs, and threads
🌀 Sweep

Extrude a 2D profile (circle, rectangle) along a 3D path (line, spline, arc). Perfect for shafts, tubes, and curved geometry.

  • Draw profile sketch (cross-section)
  • Draw path sketch
  • Sweep → Select profile and path
  • Optional: Twist angle, scale variation
🌊 Loft

Blend between two or more profiles. Creates smooth transitions between different cross-sections. Ideal for aerodynamic shapes and complex transitions.

  • Create 2+ profile sketches (circle → rectangle)
  • Loft → Select profiles in order
  • Automatic resampling for smooth blend
📊 Sheet Metal

Design flat stock and bends. cycleCAD includes 5 sheet metal operations:

  • Bend: Specify thickness, bend radius, and K-factor for flat pattern
  • Flange: Add perpendicular walls to edges
  • Tab: Small rectangular protrusions
  • Slot: Cutouts for assembly
  • Unfold: Generate flat pattern for laser cutting or waterjet
🔗 Spring Generator

Automatically create helical springs (compression, tension, torsion). Specify:

  • Outer diameter
  • Wire diameter
  • Number of turns
  • Pitch (gap between coils)
🔩 Thread Generator

Create ISO metric threads on cylinders (M5, M10, M16, etc.). Auto-generates helix geometry with proper pitch and root/crest radii per ISO 262 standard.

Pro Tip: Loft Strategy
For best results, ensure profiles have the same number of vertices and match topology. Use Sketch → Edit to reorder points if loft twists unexpectedly.

Assembly Workspace
Components, mates, joints, and exploded views

Switch to Assembly mode to combine multiple parts and constrain their positions. cycleCAD supports hierarchical assemblies with automatic BOM generation.

📦 Adding Components
  • Assembly Tab → Insert Component
  • Select .STL, .STEP, or cycleCAD .json file
  • Click 3D viewport to place
  • Edit position in properties panel
🔗 Mate Constraints

Define relationships between components:

Mate Type Constraint Use Case
Coincident Faces/planes overlap Flush assembly
Offset Parallel at distance Gasket gaps
Angle Fixed angle between faces Hinged joints
Concentric Cylinders share axis Shaft alignment
Tangent Surfaces touch smoothly Rollover contacts
💥 Exploded View

Create animation showing assembly disassembly. Set component displacement vectors and animate assembly/disassembly sequence. Export as MP4 for presentations.

Assembly Status
cycleCAD currently has beta support for assembly. Feature tree shows components but visual constraints are approximate. Full B-rep constraint solving coming in Phase B.

Text-to-CAD with AI
Natural language commands and voice input

cycleCAD includes an Agent API that interprets natural language and translates it into CAD operations. You can describe a shape in English, and the AI constructs it automatically.

🎙️ Voice Input

Click the microphone icon in the toolbar and speak a command:

  • "Create a cylinder 50mm diameter, 80mm tall"
  • "Add a hole with 10mm radius"
  • "Fillet all edges with 2mm radius"
  • "Export as STL"
Browser Requirement
Voice input requires HTTPS and microphone permissions. Test with text input first if mic isn't working.
📝 Text Commands

Type directly into the Agent panel (bottom left). Commands are parsed with NLP:

"build a rectangular boss 100x50x20mm" "add four M5 holes arranged in a circle at radius 30mm" "fillet all edges with radius 3" "create sheet metal from current sketch, bend radius 2mm, thickness 1.5mm"
🔄 Agent Pipeline (6 Steps)
  1. Understand — Parse NLP intent (create, add, modify, export)
  2. Plan — Map to CAD operations (extrude, hole, fillet, pattern)
  3. Generate — Build geometry in Three.js scene
  4. Validate — Check for DFM issues, collisions
  5. Fix — Auto-correct if needed (adjust radius, scale)
  6. Simplify — Optimize for export (reduce polygons if needed)
Pro Tip: Iterative Design
Don't try to build the entire part in one command. Start simple ("create cylinder 50") and refine: "add hole 10" → "fillet 2" → "pattern in circle 8x". The agent learns context from each step.

CAM Pipeline
From design to G-code: 8 CNC operations

cycleCAD includes a full CAM (Computer-Aided Manufacturing) module that generates tool paths and G-code for 3-axis CNC mills and routers.

🔧 8 CNC Operations
# Operation Description
1 Outline Cut 2D profile perimeter (pockets, profiles)
2 Level Face flats (reduce material height to Z-min)
3 Rough High-feedrate bulk removal (3-5mm stepdown)
4 Contour Adaptive tool path following part geometry
5 Drill Pre-drill holes before tapping or countersink
6 Trace Finish edge following (0.2-0.5mm stepdown)
7 Pocket Interior features (pockets, slots)
8 Register Flip part and machine backside
⚙️ Tool Library

Pre-configured tools with cutting speeds/feeds:

  • End mills (2-flute, 4-flute, variable)
  • Drills (stub, standard, jobber)
  • Countersinks (90°, 82°, 60°)
  • Thread mills (ISO metric, UNC)
📋 Stock Manager

Define blank size (e.g., 100x100x25mm aluminum plate) and margin. cycleCAD calculates minimum depth and recommends clamp locations.

📊 G-Code Export

Export tool paths as G-code (FANUC, Haas, Mach3 dialects). Includes:

  • Tool change comments M6 T3
  • Feedrate blocks F120 (mm/min)
  • Spindle speed M3 S1200 (RPM)
  • Tool length offset compensation H1
  • Coolant control M8/M9
Tip: Verify Before Cutting
Always simulate G-code in CAM software (Fusion 360, Carbide Create) before loading on CNC. Verify tool paths don't collide with fixtures or part.

Section Views & Cross-Sections
Cut planes, 2D projection, and technical drawings

Section cuts reveal interior geometry. cycleCAD uses WebGL clipping planes to dynamically cut the 3D model and extract 2D engineering drawings.

✂️ Cut Planes

Select a plane orientation:

  • XY (Top) — Horizontal cut, shows plan view
  • XZ (Front) — Vertical cut, shows elevation
  • YZ (Side) — Vertical cut, shows side profile
  • Custom — Arbitrary plane angle (for sectional views through features)
📐 Section Views in Drawings

Combine multiple sections (A-A, B-B) in engineering drawings. cycleCAD auto-generates hidden line rendering and dimension lines.

📤 Export to 2D
  • DXF — AutoCAD format, suitable for further editing
  • SVG — Vector graphics, embeddable in web
  • PDF — High-quality technical drawing output
Current Limitation
Automatic section generation is in beta. Manual section placement currently requires manual sketching in 2D view. Parametric sections coming in Phase C.

AI Chatbot & Design Assistant
Gemini + Groq + offline fallback for CAD guidance

cycleCAD includes a hybrid AI chatbot that understands CAD context. Ask questions about manufacturability, material selection, cost, or assembly strategy.

💬 Chat Features
  • Query current part geometry (volume, surface area, weight)
  • Material recommendations based on use case
  • Manufacturing process suggestions (CNC, 3D print, injection molding)
  • Assembly sequence advice
  • Cost estimation based on material and machining time
🔌 AI Backends

cycleCAD uses a 3-tier fallback system:

  1. Gemini Flash API (Primary) — Google's fastest model, 1M context window
  2. Groq Llama 3.1 8B (Fallback) — Open-source, ultra-low latency (~30ms)
  3. Offline NLP (Offline Mode) — Pattern matching when no internet
Pro Tip: CAD-Aware Queries
Be specific: "I have an aluminum part with 500cm² surface area and thin walls 1.5mm. Can I CNC mill it or should I 3D print?" The chatbot analyzes your current part geometry.

DFM Analysis
Design for Manufacturability & Cost Estimation

DFM (Design For Manufacturing) analysis checks your design against practical manufacturing constraints and estimates production cost.

🔍 Automated DFM Checks
  • Wall thickness: Min 0.8mm for injection molding, 0.5mm for CNC aluminum
  • Draft angle: Recommend 2-5° for molded parts
  • Undercuts: Flag geometry that requires side action cores
  • Fillet radius: Warn if fillet < half wall thickness
  • Hole alignment: Detect holes that would require complex tooling
  • Thin features: Alert if feature width < 2× depth (weak geometry)
💰 Cost Estimation

Select material and process for real-time cost breakdown:

Process Material Cost Labor/Machine Time Tooling
CNC Aluminum Per kg: ~$5-10 $80-150/hr $0 (no setup)
3D Print (FDM) Per kg: ~$15-30 $2-5/hr $0
Injection Molding Per kg: ~$2-5 $1-3/unit (high vol) $5K-50K per mold
Sheet Metal Per kg: ~$3-8 $50-100/hr $500-2K (dies)
📊 Material Selection

cycleCAD includes libraries for 15+ materials:

  • Metals: Aluminum 6061-T6, Steel, Stainless, Titanium, Copper
  • Plastics: ABS, PLA, PETG, Nylon, Polycarbonate
  • Composites: Carbon fiber, Fiberglass
Tip: Iterate on Cost
Use DFM to make quick design decisions. E.g., "Injection molding would save 40% at 1000 units, but CNC is $80 setup-free for prototypes." Edit sketch, re-cost, repeat.

Import & Export Workflows
STEP, STL, DXF, Inventor, and reverse engineering

cycleCAD reads and writes industry-standard file formats. Import CAD from other tools, or reverse-engineer from 3D scans and STL files.

📥 Import Formats
Format What It Does Typical Use
.STEP Parametric B-rep geometry (solid geometry) Exchange with SolidWorks, Fusion, OnShape
.STL Triangle mesh (non-parametric) 3D print models, reverse engineering
.IPT / .IAM Autodesk Inventor parts and assemblies Migrate from Inventor to cycleCAD
.DXF 2D drawing geometry Import sketches from AutoCAD, LibreCAD
cycleCAD .JSON Native cycleCAD format (fully parametric) Save/load cycleCAD designs
📤 Export Formats
Format Resolution Use Case
.STL (Binary/ASCII) Mesh (0.1-1mm triangles) 3D printing, CNC simulation
.OBJ Mesh (with texture coordinates) 3D visualization, rendering
.glTF 2.0 Mesh + PBR materials Web preview, AR/VR, game engines
.DXF (2D/3D) Wireframe or 3D projection CAM software, AutoCAD, laser cutting
.G-Code Tool paths for CNC CNC mills, routers, waterjet
🔄 Reverse Engineering from STL

Import an STL (from 3D scan, competitor product, or legacy CAD) and automatically generate parametric sketches and features:

  1. Detect flat faces → sketches
  2. Identify cylinders and holes
  3. Extract draft angles
  4. Build feature history (extrude, fillet, pattern)
Reverse Engineering Caveat
Automatic feature detection is ~80% accurate. You may need to manually refine sketches or recreate complex features. Use as a starting point, not a complete replacement.
📦 Inventor Project Loading

Load entire Inventor projects (.ipj). cycleCAD parses folder structure, finds all parts (.ipt), assemblies (.iam), and generates a Bill of Materials:

  • Browse .ipj projects in file picker
  • View component tree with quantities
  • Export BOM to CSV for purchasing
  • Identify custom vs. standard parts
