Platform Architecture

⚙️ CAD Engine (Core)

  • Microkernel (kernel.js) — Hot-swap module system
  • 45+ JS Modules — Measurements, notes, analysis, threads, fasteners, materials, workflows
  • Three.js r170 — 3D rendering with WebGL 2.0
  • Constraint Solver — 12-type 2D constraint relaxation
  • B-Rep Kernel — OpenCascade.js WASM (STEP import/export)

🤖 Agent API (55 Commands)

  • 10 Namespaces: shape, feature, assembly, render, validate, export, import, material, analysis, admin
  • JSON-RPC Transport — Stdio (MCP), HTTP, WebSocket, Webhooks
  • Self-Describing Schema — Introspection for AI agents
  • Native Bindings — Direct module access for in-browser use
  • Cloud-Ready — All operations remote-callable

📡 Integration Layer

  • MCP Server (1,161 lines) — Claude integration
  • REST API (1,120 lines) — HTTP endpoints + WebSocket
  • CLI Tool (662 lines) — Terminal REPL + batch mode
  • Webhooks — Event streaming for fab networks
  • SDK — npm package: cyclecad v3.2.1

💰 Token Engine ($CYCLE)

  • Per-Op Billing — Extrude: 50 tokens, Revolve: 75, Fillet: 150
  • Double-Entry Ledger — Spend (user) + Earn (creator)
  • 3 Tiers: FREE (10/day), PRO (€49/mo), ENTERPRISE (€299/mo)
  • Batch Discounts — 25-50% for multi-op scans
  • Creator Royalties — 70-90% for marketplace models

🏪 Model Marketplace

  • 5-Stage Pipeline — Publish → Review → Listed → Sales → Royalties
  • 7 Access Tiers — Free, PRO, Subscription, Custom, Exclusive, OEM, Educational
  • 8 Demo Models — Gear, Washer, Bolt, Bracket, Frame, Electronics, Assembly, Spring
  • Creator Dashboard — Analytics + payout management
  • Revenue Split: 70% Creator / 30% Platform

🔍 ExplodeView (20,900+ lines)

  • Standalone Viewer — 399 parts, 6 assemblies, 63+ features
  • AI Vision ID — Gemini Vision API + geometry fallback
  • Batch Scanning — Auto-analyze all parts with progress bar
  • Smart BOM — AI-identified parts with pricing + order links
  • Maintenance Heatmap — Color-code by urgency, export schedule

🧠 AI Integration (3-Tier)

  • Cloud LLM: Gemini Flash (Fast, multimodal) + Groq Llama 3.1 8B (Low-cost inference)
  • Local AI: Ollama + DistilBERT on-device fallback
  • Offline NLP: RxDB-based semantic search, no network needed
  • API Keys: localStorage secure storage with encryption
  • Cost Optimization: Prompt caching + batch processing

📊 Data & State Management

  • Global Scope: window.allParts, window.ASSEMBLIES, window.manifest (ExplodeView)
  • Persistence: IndexedDB (models) + localStorage (prefs)
  • Real-time: WebSocket events from signaling server
  • Snapshots: Save/load CAD states as JSON
  • Git-Like Versioning: Full history with visual diff

🌐 Deployment

  • GitHub Pages: exploreview.com + cyclecad.com (DNS CNAME)
  • Docker Compose: Local dev + staging (port 8080)
  • npm Packages: cyclecad v3.2.1, explodeview v1.0.17
  • Service Workers: Cross-Origin Isolation (COOP/COEP)
  • HTTPS: GitHub Pages auto-enabled, custom domains supported

Data Flow

User Input (UI/Voice/API) ↓ Agent API / MCP / REST ↓ CAD Engine (Microkernel) ↓ Module System (Sketch → Ops → Export) ↓ Three.js Viewport (Render) ↓ IndexedDB / localStorage (Persist) ↓ Signaling Server (Collab) ↓ Fab Network / Stripe / Agent Swarms
