Data Flow: Human vs Agent

🖥️ Human Interface Flow

User clicks "Extrude" button in toolbar
→ DOM event listener fires
→ operations.extrude(params)
→ Three.js geometry updated in scene
→ Feature tree panel re-rendered
→ Undo history snapshot saved
→ User sees result in 3D viewport

🤖 Agent Interface Flow

Agent sends: { method: "feature.extrude", params: { depth: 50 } }
→ agent-api.js dispatches command
→ operations.extrude(params)
→ Three.js geometry updated in scene
→ Feature tree panel re-rendered
→ Undo history snapshot saved
→ Agent receives: { ok: true, result: { entityId: "extrude_1" } }

Both paths call the SAME operations.extrude() — one codebase, two interfaces.
